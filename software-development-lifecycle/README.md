# SDLC Framework

A lean software development lifecycle for a small team (4 developers, 1 QA, 1 BA, 1 part-time PO) working on a project.

## Quick Start

→ **New to the framework?** Start with [getting-started.md](getting-started.md)

## Phases

| Phase | Cheat Sheet | What You Produce |
|-------|-------------|-----------------|
| 1. Define | [phases/1-define.md](phases/1-define.md) | Business Case, Project Charter, BRD, SRS |
| 2. Design | [phases/2-design.md](phases/2-design.md) | ADD, ADRs, DDD, API Spec, Data Model, Diagrams |
| 3. Build & Verify | [phases/3-build-and-verify.md](phases/3-build-and-verify.md) | Code, PRs, Test Plan, Test Cases, UAT Sign-Off |
| 4. Ship & Run | [phases/4-ship-and-run.md](phases/4-ship-and-run.md) | Deployment, Release Notes, Monitoring |
